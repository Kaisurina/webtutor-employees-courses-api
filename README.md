# WebTutor Integration Project

Проект тестового задания с WebTutor для управления сотрудниками.

## Описание

Проект состоит из двух компонентов:

1. Веб-шаблон (контроллер) - REST API
2. Агент уведомлений - автоматические напоминания

## Структура

- core.types.ts - основные типы данных
- controllers/ - веб-контроллеры
- agents/ - агенты WebTutor
- Task.md - техническое задание

## Core Types

Центральный файл с типами и утилитами.

### WebTutor объекты

- CurRequest - объект запроса
- CurResponse - объект ответа
- tools - инструменты WebTutor
- log - система логирования

### Утилиты

Безопасность:

- escapeSQLString() - экранирование SQL
- validateNumericId() - валидация ID

HTTP ответы:

- sendJSON() - отправка JSON
- sendError() - отправка ошибки

Логирование:

- logWithPrefix() - логи с префиксом

## Компоненты

### Controllers

REST API для работы с сотрудниками.
Подробности в controllers/README.md

Функции:

- Список подчиненных с поиском
- Карточка сотрудника
- Курсы сотрудника
- Назначение курсов

### Agents

Автоматические уведомления.
Подробности в agents/README.md

Функции:

- Ежедневные напоминания
- Группировка по сотрудникам
- Настраиваемые шаблоны

## Установка

### Требования

- WebTutor 4.2+
- Microsoft SQL Server

### База данных

1. Выполнить agents/indexes.sql
2. Проверить таблицы collaborators, active_learnings, learnings

### Развертывание

Контроллер:

1. Разместить в папке веб-шаблонов
2. Создать веб-шаблон в WebTutor
3. Настроить маршрутизацию

Агент:

1. Импортировать как агент
2. Настроить расписание
3. Активировать в системе

## API

Базовый URL: /template/?action=action_name
Аутентификация: через контекст WebTutor
