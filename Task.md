# Задача 1

Реализовать веб шаблон (контроллер с набором эндпоинтов) для выполнения задач по работе со списком сотрудников.
Результатом задачи должен быть пакет с веб-шаблоном или файл с кодом.

## Основные требования

- Нужны эндпоинты для выполнения следующих действий:1. Получение списка подчинённых сотрудников

1. Эндпоинт должен поддерживать пагинацию и поиск по ФИО сотрудника.
   Поиск подчинённых сотрудников должен производиться по func_managers.
   Возвращать нужно массив Collaborator.
   Схема Collaborator:

```ts
type Collaborator = {
  id: string; // id карточки сотрудника
  fullname: string; // ФИО сотрудника
  position_name: string; // Название должности
  is_dismiss: boolean; // Флаг уволен или нет
};
```

2. Получение данных по сотруднику
   Получение данных по id сотрудника
   Возвращать нужно запись CollaboratorCard.
   Схема CollaboratorCard:

```ts
type CollaboratorCard = {
  id: string; // id карточки сотрудника
  fullname: string; // ФИО сотрудника
  position_name: string; // Название должности
  is_dismiss: boolean; // Флаг уволен или нет
  avatar_url?: string; // url аватарки
};
```

3. Получение списка назначенных и пройденных курсов сотруднику
   Получение данных по id сотрудника.
   Возвращать нужно массив Row.
   Схема Row:

```ts
type Row = {
  id: string; // id карточки learnings или active_learnings
  type: "active_learning" | "learning"; // тип назначения
  state_id: number; // id статуса
  course_id: string; // id курса
  course_name: string; // Название курса
};
```

4. Назначение курса сотруднику
   Назначение производится по 2 параметрам - collaborator_id и course_id.Возвращать нужно id карточки незавершённого курса.

## Доп требования:

- Все ответы должны быть в формате JSON
- Все запросы в БД должны быть написаны на SQL
- Ошибки на клиент должны уходить с HTTP статусами

# Задача 2

Разработать агент, который будет раз в сутки в 8 утра собирать информацию по сотрудникам, у которых есть незавершённые курсы или тесты и отправлять им уведомление-напоминание о незавершённом обучении.

## Основные требования:

- В выборку должны попасть только НЕ уволенные сотрудники
- Шаблон уведомления нужно продумать самостоятельно. Основное требование, чтобы уведомление было отправлено одно и в нём содержалась информация по не завершённым курсам и тестам.
- Результатом по задаче должен быть пакет с агентом, типом уведомления и шаблоном уведомления.
